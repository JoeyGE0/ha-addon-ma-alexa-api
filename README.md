# Music Assistant Alexa API Bridge Add-on

Minimal Home Assistant add-on wrapper for the Music Assistant Alexa API Bridge.

## Configuration

- `username`: Basic auth username (default: admin)
- `password`: Basic auth password (default: test)

## Usage

1. Add this repository to Home Assistant add-on store
2. Install the add-on
3. Configure username/password in the add-on options
4. Start the add-on
5. Access the API bridge on port 3000
